# 半导体 EDA / IP 产业链深挖

> serenity-skill 产业链卡点扫描。研究日期:2026-06-26(返工加深版)。数据 Eastmoney/腾讯一手口径(财务序列/分产品/十大流通股东/资产负债/研发)。仅研究支持,非买卖建议。

## 一句话判断

EDA(芯片设计工具)是半导体最"软"也最卡的一层——新思/楷登/西门子三家 ~80%+,且受美国出口管制直接波及(既是威胁、也是国产替代催化)。**返工后核心纠正:华大九天归母 -44%、净利仅 4.6%,根因是研发砸到营收的 64.8%**——毛利高达 89%(软件特性),它是把几乎所有毛利再投进研发去追"数字全流程"。所以是"**真卡点 + 极致研发投入 + 国资硬撑**",不是生意恶化;但数字全流程仍未突破,这是最卡的一关。

## 二、本链头号卡点:华大九天(301269)🎯
- **卡住的环节**:国产 EDA 龙头,**模拟电路全流程国产唯一**;数字/制造类点工具齐全。**分产品(FY2025):EDA 软件 10.75 亿(81.1%)+ 技术服务 2.01 亿(15.2%)**;毛利 89.3%(软件)。
- **财务"为什么 -44%"(返工核心)**:营收 13.25 亿 +8%,**研发费 8.59 亿、占营收 64.8%(!)**——把净利压到 4.6%(归母 0.61 亿、-44%)。这是**砸钱追数字全流程的极致投入**,不是生意垮;但也说明**它离全流程突破还远、要持续烧**。
- **财务质量/治理**:资产极轻(存货 0.5 亿、软件公司)、货币 8.1 亿;股东:**中国电子(CEC)21.1% + 九创汇新 14.9% + 中电金投 12.8% + 大基金一期 8.3%**——**CEC(中国电子)系 + 大基金,标准国家队 EDA 旗舰**。
- **主要风险(重点划线)**:**数字全流程未突破(最卡一关)**,64.8% 研发要长期烧、利润短期难起;PE 极高(失真);美国 EDA 出口管制是双刃(对华为/先进节点是威胁,对国产替代是催化);点工具拼凑 ≠ 全流程平台。

## 三、另几柱
- **概伦电子(688206)**:器件建模/制造类 EDA,PE 835x;**广立微(301095)**:良率/测试 EDA + 测试机,PE 220x——均点工具、规模小、估值高。
- **芯原股份(688521)⚠️**:IP+一站式定制(chiplet)。**分产品:量产业务 14.90 亿(47%,毛仅 18%)+ 芯片设计 8.77 亿(28%,毛 14%)+ IP 授权 6.71 亿(21%,毛 85.8%)**——**真正高毛利的只有 IP 授权(21%),量产/设计是低毛利**,拖累整体**归母 -5.28 亿(亏)、PB 56x(全场最贵)**。IP 国产稀缺,但盈利结构差、估值极端。

## 四、反方 / 市场没看清
1. **卡点真、利润被研发吃光**(华大 64.8% 研发);多头赌"研发→全流程突破→利润释放",空头看"数字全流程遥遥无期+估值离谱";2. 芯原高毛利只有 IP 那 21%,量产/设计拉低到巨亏;3. 出口管制双刃;4. 全行业熔涨。
